<h2> Getting Started</h2>

1. Modify the variable url from the file NasaAPI.js with NASA URL and your key, that you can get here: https://api.nasa.gov/

2. Install dependencies

   ```bash
   npm install
   ```

3. Start it up

   ```bash
   npm start
   ```
