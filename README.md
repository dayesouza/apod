## Getting Started

1. Copy the `.env.EXAMPLE` file to a `.env` file and fill in the environment variable from your Nasa Key that you can get here: https://api.nasa.gov/

2. Install dependencies

   ```bash
   npm install
   ```

3. Start it up

   ```bash
   npm start
   ```
